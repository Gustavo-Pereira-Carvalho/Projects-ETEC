Arquivos de Video

AVI

extensão de arquivo AVI criado por Microsoft e é usado para armazenar dados de vídeo e áudio. arquivo AVI é suportado por vários programas de media player.

Fonte

MPG

MPG arquivo contém os filmes que são distribuídos sobre o arquivo internet.MPG contém vídeos compactados para habilitar o usuário para baixar os arquivos a partir da Internet mais rapidamente e com boa ficheiro too.MPG qualidade também suporta vídeos VCR que são convertidos em filmes para download

Fonte

WMV

WMV arquivo é um arquivo de vídeo que contém vídeo codificado com um dos Windows Media Vídeo codecs proprietários da Microsoft. WMV arquivo usa o formato do recipiente Advanced Systems Format (ASF) para encapsular o conteúdo multimídia codificado.

Fonte

MOV

QuickTime formato multimídia pela Apple. Ele é suportado no sistema Mac e Windows. Ele pode ter um vídeo comprimido ou faixa de áudio
