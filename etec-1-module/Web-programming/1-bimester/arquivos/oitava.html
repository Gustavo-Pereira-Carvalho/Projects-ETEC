ZIP

Zip é um formato de arquivo amplamente utilizado que é usado para compactar um ou mais arquivos juntos em um único local, reduzindo o tamanho geral e facilitando o transporte dos arquivos. Arquivos zip funcionam de modo muito similar a uma pasta padrão em seu computador. No entanto, ao colocar os arquivos no formato zip, o conteúdo da pasta é compactado, reduzindo a quantidade de dados usados pelo computador.
