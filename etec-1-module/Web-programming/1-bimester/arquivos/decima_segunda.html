Arquivos de Pagina

JSP

A extensão do arquivo JSP como a extensão ASP ou PHP arquivo é uma página web gerado por um servidor web. Mas difere em que contém um código java em vez de ActiveX ou PHP. Este código é analisado por este servidor web, que produz HTML que é enviado para o computador dos usuários. E, em resultado da análise deste código pelo servidor web, a JSP código nunca aparece para o usuário final, mas apenas o HTML produzido pelo código Java na página. Um editor de texto básico ou um programa de desenvolvimento web pode editar os JSP páginas.

CFM

O arquivo CFM é um arquivo ColdFusion que consiste em ColdFusion Markup Language (CFML) ou código ColdFusion. É normalmente aberto por um usuário ao abrir uma página através de um servidor web.

SWF

Um arquivo SWF é um arquivo de animação, principalmente associado com Adobe Flash. Pode ser jogado pelo Adobe Flash Player ou um navegador web, enquanto um plugin do Flash está instalado. Foi concebido como um formato de entrega eficiente e não como um formato para troca de gráficos entre editores gráficos. jogos de vídeo baseados na web muitas vezes usam o SWF formato de arquivo.
